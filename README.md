# KaomojiCLI

Python CLI kaomoji advisor  

## Getting started

### Prerequisites
- Python 3.6 or newer

### Install
```bash
python setup.py install
```

### Usage

1. `kaomoji` or `kaomoji-cli` to start CLI
2. Search for category you want
3. Press `Enter` and your kaomoji is already in clipboard!
4. `Ctrl+C` or `Ctrl+D` to quit
